# cli.okta.com redirect

This repo previously hosted the Okta CLI documentation site (Jekyll, served via Netlify). The site has been retired — `cli.okta.com` now permanently redirects to [developers.okta.com](https://developers.okta.com).

The redirect is defined in [`vercel.json`](./vercel.json) and deployed on Vercel. See git history prior to this change for the original Jekyll site source.
